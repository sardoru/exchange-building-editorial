# The House That Cotton Built

Editorial feature on the Memphis Merchants' & Cotton Exchange Building (1910), and its new life as a node for AI and on-chain trust.

**Live:** _deploy URL added after `vercel --prod`._

## Stack
- Static HTML/CSS/JS (no build step)
- Google Fonts: Fraunces, Newsreader, Space Mono
- Deployed on Vercel as a static site

## Local preview
```bash
python3 -m http.server 8000
# open http://localhost:8000
```
